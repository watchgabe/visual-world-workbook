**2. Create `src/components/auth/LoginForm.tsx`:**

Mark as `'use client'`. This is the core client component.

**State:**
- `email: string` (controlled input)
- `sent: boolean` (toggles between form and success message)
- `loading: boolean` (spinner state for D-12)
- `errorMessage: string | null` (display errors from URL params or send failures)
- `secondsLeft: number` (30-second countdown for D-04)
- `showResend: boolean` (appears after countdown)

**Error message map (D-11, D-13):**
```typescript
const AUTH_ERRORS: Record<string, string> = {
  link_expired: "Hmm, that link has expired. Enter your email to get a new one.",
  unknown: "Something went wrong. Please try again.",
  rate_limit: "Too many requests. Please wait a minute before trying again.",
}
```

**Props:** Accept `redirectPath: string` (the redirect param from URL, default `'/'`) and `errorParam: string | null` (from URL search params).

**On mount:** If `errorParam` exists, set `errorMessage` from AUTH_ERRORS map.

**Send magic link handler:**
- Set `loading = true`, clear `errorMessage`.
- Call `supabase.auth.signInWithOtp({ email, options: { emailRedirectTo: \`\${window.location.origin}/auth/callback?next=\${encodeURIComponent(redirectPath)}\` } })` — per D-06/D-07.
- On error: set `errorMessage` based on error.message (map rate limit errors to friendly text).
- On success: set `sent = true`, start 30-second countdown timer.
- Set `loading = false`.

**Resend timer (D-04):** When `sent` becomes true, start an interval decrementing `secondsLeft` from 30. When it hits 0, set `showResend = true` and clear interval. Cleanup on unmount.

**Resend handler:** Reset `sent = false`, `showResend = false`, `secondsLeft = 30`, then call send again.

**Render — Form state (sent === false):**
- Email input: styled with CSS vars (background: var(--card), border: var(--border), color: var(--text), border-radius: var(--radius-md), padding: 12px 16px, font-size: 15px, width: 100%).
- Submit button: background var(--orange), color white, border-radius var(--radius-md), padding 12px, font-weight 600, width 100%. On hover: var(--orange-hover). When loading: show a spinner (simple CSS border-spin animation) and disable button (per D-12).
- Error message if errorMessage is set: text in a warm tone, styled with color var(--orange-dark) on a subtle background.

**Render — Success state (sent === true, D-03):**
- Hide the form.
- Show a mail/check icon (use an inline SVG envelope icon).
- Heading: "Check your email" in var(--text).
- Subtext: "We sent a magic link to {email}. Click the link in the email to sign in." in var(--dim).
- If `showResend`: Show a "Resend magic link" button styled as text link in var(--orange).
- If not `showResend` and secondsLeft > 0: Show "Resend available in {secondsLeft}s" in var(--dimmer).
